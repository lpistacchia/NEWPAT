# NEWPAT

NEWPAT is a collection of downstream analysis modules developed for cfDNA-based paternity testing using Double Nucleotide Polymorphisms (DNPs).

The repository contains tools for fetal fraction estimation, Combined Paternity Index (CPI) calculation, contamination assessment, SNP calling, and methodological evaluation of DNP error rates. The modules are designed to operate on data generated by the DNPcall workflow and can be used independently according to the specific analytical objective.

## Repository Structure

### CPI_from_cffDNA

This module estimates the fetal fraction using maternal-informative DNPs and computes Combined Paternity Index (CPI) distributions across increasing fetal-read thresholds. Starting from the per-site counts generated by DNPcall for mother and putative father samples, it produces pair-specific CPI plots and summary tables.

See `CPI_from_cffDNA/README.md` for detailed instructions.

### contamination

Tools for contamination assessment and related quality-control analyses.

See `contamination/README.md` for detailed instructions.

### SNPcall

SNP-calling workflow adapted from DNPcall while preserving the original filtering criteria. This allows the same genomic positions to be consistently reused for downstream SNP-based analyses, including chromosome Y investigations and single-base CPI calculations.

See `SNPcall/README.md` for detailed instructions.

### DNP_error_rate

Standalone module used to estimate single-base and double-base error rates in DNP sequencing data. This analysis was used to characterize the error profile of the assay and support the methodological framework described in the associated publication.

See `DNP_error_rate/README.md` for detailed instructions.

## Main Application

The primary application of NEWPAT is the estimation of fetal fraction and the calculation of CPI distributions from DNPcall output files through the `CPI_from_cffDNA` module, with optional contamination assessment using the `contamination` tools.

Additional modules (`SNPcall` and `DNP_error_rate`) are provided for specific analytical and methodological purposes and can be used independently of the main cfDNA paternity-testing workflow.
